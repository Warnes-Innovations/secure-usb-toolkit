🔐 Secure USB Drive

This USB drive contains a password-protected encrypted archive for securely storing files. The archive (secure_data.7z) uses AES-256 encryption via 7-Zip, a free and open-source tool available on every major platform.

A portable copy of PeaZip (a free, open-source file manager that includes 7-Zip support) is bundled on this drive for your platform — no separate download required.

⚠️ Important: If you forget your password, your data cannot be recovered — not even by the person who gave you this drive. Keep your password safe.

1 Open the Encrypted Archive

Use the bundled PeaZip portable or any 7-Zip-compatible tool already installed on your computer. Choose the instructions for your platform below.

🪟 Windows

  1. Open the PeaZip-Windows folder on this USB drive and run PeaZip.exe — no installation needed.
  2. Use File → Open archive… (or drag and drop) to open secure_data.7z from the root of this USB drive.
  3. Enter your password when prompted.
  4. Select the files or folders you want and click Extract to save them to your computer.

Alternatively, use 7-Zip if already installed: right-click secure_data.7z → 7-Zip → Extract files…

🍎 macOS — Intel

  1. Locate PeaZip-macOS-Intel.zip on this USB drive and double-click to unzip it. Move the resulting PeaZip.app to your Desktop or Applications folder.
  2. First launch only: right-click (or Control-click) the app and choose Open to bypass Gatekeeper, then click Open again in the dialog.
  3. Open secure_data.7z from the root of the USB drive via File → Open archive…, enter your password, and extract your files.

🍎 macOS — Apple Silicon (M-series)

  1. Locate PeaZip-macOS-ARM.dmg on this USB drive and double-click to mount it. Drag PeaZip.app to your Applications folder.
  2. First launch only: right-click the app and choose Open, then confirm in the Gatekeeper dialog.
  3. Open secure_data.7z from the USB drive, enter your password, and extract your files.

Not sure which Mac you have? Click the Apple menu → About This Mac. If it says Apple M…, you have Apple Silicon. If it says Intel, use the Intel installer above.

🐧 Linux — x86-64

  1. Open the PeaZip-Linux-x86 folder on this USB drive.
  2. Run peazip from that folder (you may need to mark it executable first: chmod +x peazip).
  3. Open secure_data.7z, enter your password, and extract your files.

Alternatively: 7z x secure_data.7z if 7-Zip is already installed (sudo apt install 7zip / sudo dnf install 7zip).

🐧 Linux — ARM64 (aarch64)

  1. Open the PeaZip-Linux-ARM folder on this USB drive.
  2. Run peazip from that folder (chmod +x peazip if needed).
  3. Open secure_data.7z, enter your password, and extract your files.

2 Safely Remove the Drive When Done

Because the archive is a regular file (not a mounted volume), there is nothing to dismount — simply close PeaZip and eject the USB drive normally.

  1. Close PeaZip or any other application that has files from the USB open.
  2. Eject the drive from your operating system before unplugging it (right-click the drive in Explorer / Finder / file manager → Eject).
